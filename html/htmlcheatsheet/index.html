HTML CheatSheet

  • HTML is an acronym which stands for Hyper Text Markup Language which is used for creating web pages and web applications. Let's see what is meant by Hypertext Markup Language, and Web page.
  • Hyper Text: HyperText simply means "Text within Text." A text has a link within it, is a hypertext. Whenever you click on a link which brings you to a new webpage, you have clicked on a hypertext. HyperText is a way to link two or more web pages (HTML documents) with each other.
  • Markup language: A markup language is a computer language that is used to apply layout and formatting conventions to a text document. Markup language makes text more interactive and dynamic. It can turn text into images, tables, links, etc.
  • Web Page: A web page is a document which is commonly written in HTML and translated by a web browser. A web page can be identified by entering an URL. A Web page can be of the static or dynamic type. With the help of HTML only, we can create static web pages.
  • Hence, HTML is a markup language which is used for creating attractive web pages with the help of styling, and which looks in a nice format on a web browser. An HTML document is made of many HTML tags and each HTML tag contains different content.

<title>

The title tag is an HTML code tag that allows you to give a web page a title. This title can be found in the browser title bar, as well as in the search engine results pages (SERP). It's crucial to add and optimise your website's title tags, as they play an essential role in terms of organic ranking (SEO).

Example:- 


<!DOCTYPE html>
<html lang="en">
<head>
<title>HELLO HTML</title>
</head>
<body>
<h1>This is heading</h1>
</body>
</html>

The <div> tag is easily styled by using the class or id attribute.
Any sort of content can be put inside the <div> tag!

</body>

HTML Interview Question

Question 1: What is the purpose of the DOCTYPE declaration in HTML?
Question 2: What is the difference between <div> and <span> elements in HTML?
Question 3: What is HTML?
Question 4:What are the basic building blocks of HTML?
Question 5:What is the difference between HTML and CSS?
Question 6: Explain the difference between inline and block-level elements.
Question 7:What is the purpose of the "alt" attribute in the "img" tag?
Question 8:What is semantic HTML?
Question 9:How do you create a hyperlink in HTML?
Question 10:Explain the concept of forms in HTML
Question 11:What is the purpose of the "meta" tag in HTML?
Question 12:How do you comment in HTML?
Question 13:What is the difference between the "span" and "div" elements?

Answer 1: The DOCTYPE declaration (Document Type Declaration) specifies the type and version of the HTML being used in a document. It helps the browser interpret the document correctly by indicating the HTML version and the rules that should be followed for rendering Example DOCTYPE declaration for HTML5: <!DOCTYPE html>

Answer 2: Both <div> and <span> are used as container elements, but they are used for different types of content. <div> is a block-level element that is used to group and structure larger sections of content, while <span> is an inline-level element used for smaller portions of text or elements within a line of text.

Answer 3: HTML stands for HyperText Markup Language. It is the standard markup language used to create and structure content on the web.

Answer 4: The basic building blocks of HTML are elements, which are represented by tags. Tags are enclosed in angle brackets and consist of an opening tag, content, and a closing tag.

Answer 5: HTML is used to structure the content on a web page, while CSS (Cascading Style Sheets) is used to control the presentation and layout of the content.

Answer 6: Inline elements are those that do not start on a new line and only take up as much width as necessary. Block-level elements, on the other hand, start on a new line and take up the full width available.

Answer 7: The "alt" attribute in the "img" tag is used to provide alternative text for the image, which is displayed if the image cannot be loaded or if the user is using a screen reader.

Answer 8: Semantic HTML involves using HTML elements that carry meaningful information about the structure and content of the page. It helps search engines, accessibility tools, and developers better understand the content.

Answer 9: Hyperlinks are created using the "a" (anchor) tag. The "href" attribute within the tag specifies the URL to which the hyperlink points.

Answer 10: Forms are used to collect user input. They are created using the "form" tag and contain various input elements like text fields, radio buttons, checkboxes, and buttons.

Answer 11: The "meta" tag is used to provide metadata about the HTML document, such as character encoding, author, description, and keywords.

Answer 12: Comments in HTML are created using the <!-- comment --> syntax. They are ignored by the browser and are often used to provide notes to developers within the code.

Answer 13: The "span" element is an inline container used to style and manipulate small sections of text or content. The "div" element is a block-level container used to group larger sections of content for styling or layout purposes.

Remember that interview questions can vary in difficulty and focus, so it's important to have a solid understanding of HTML fundamentals and possibly some more advanced topics depending on the job position.
